# 📋 Weekly Task Manager

ระบบจัดลำดับงานรายสัปดาห์ที่พัฒนาด้วย HTML, CSS และ JavaScript

ช่วยจัดการงานตามวันและเวลาที่กำหนดส่ง พร้อมแสดงระดับความเร่งด่วน เพื่อให้วางแผนและติดตามงานได้ง่ายขึ้น

## ✨ Features

✅ จัดเรียงงานตามวันและเวลาส่งอัตโนมัติ

✅ แสดงระดับความเร่งด่วนของงาน
- 🔴 ด่วน (ภายใน 24 ชั่วโมง)
- 🟠 ปานกลาง (1–3 วัน)
- 🟢 ไม่เร่งด่วน (มากกว่า 3 วัน)

✅ จัดกลุ่มงานตามวันกำหนดส่ง

✅ เพิ่มงานใหม่พร้อมวันและเวลา

✅ ลบงานที่ไม่ต้องการ

✅ แนบไฟล์กับแต่ละงานได้

✅ แสดงเวลาส่งของแต่ละงาน

✅ Responsive Design รองรับมือถือและเดสก์ท็อป

✅ Custom Favicon

✅ Online Deployment ด้วย GitHub Pages และ Netlify
