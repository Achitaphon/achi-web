# 📋 Weekly Task Manager

เว็บจัดลำดับงานรายสัปดาห์ พัฒนาโดยใช้ HTML, CSS และ JavaScript

## ✨ Features

- จัดเรียงงานตามวันและเวลาอัตโนมัติ
- เพิ่มและลบงานได้
- รองรับ Dark Mode
- บันทึกข้อมูลใน Local Storage
- รองรับการใช้งานบนมือถือ
